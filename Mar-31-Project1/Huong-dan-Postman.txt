====================================================
  HUONG DAN POSTMAN - MESSAGE ROUTES
  Base URL: http://localhost:3000/api/v1/messages
====================================================

SCHEMA MESSAGE:
- from: user
- to: user
- messageContent: {
    type: String[file,text]
    text: String
  }

LUU Y:
- Route messages can dang nhap (token)
- Header:
    Key: Authorization
    Value: <token>

====================================================
1. GET /:userID
====================================================
Muc dich:
- Lay toan bo message giua user hien tai va userID
- Gom 2 chieu:
  + from = user hien tai, to = userID
  + from = userID, to = user hien tai

Postman:
- Method: GET
- URL: {{base}}/<userID>
- Headers: Authorization: <token>

====================================================
2. POST /
====================================================
Muc dich:
- Gui message den userID
- Neu co file:
  + messageContent.type = "file"
  + messageContent.text = duong dan file
- Neu la text:
  + messageContent.type = "text"
  + messageContent.text = noi dung gui

Postman (gui text):
- Method: POST
- URL: {{base}}
- Headers: Authorization: <token>
- Body: form-data
  + to: <userID>
  + text: Xin chao

Postman (gui file):
- Method: POST
- URL: {{base}}
- Headers: Authorization: <token>
- Body: form-data
  + to: <userID>
  + file: [chon file]

====================================================
3. GET /
====================================================
Muc dich:
- Lay message cuoi cung cua moi user ma:
  + user hien tai da nhan tin den hoac
  + user khac da nhan tin den user hien tai

Postman:
- Method: GET
- URL: {{base}}
- Headers: Authorization: <token>
